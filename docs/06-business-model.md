# 비즈니스 모델

## 수익 구조

### 기본 모델: 거래 수수료

| 항목 | 비율 |
|------|------|
| **플랫폼 수수료** | 5% |
| PG 수수료 (토스페이먼츠 등) | ~2.8% + α |
| **크리에이터 수령** | ~92% |

### 수수료 계산 예시

**후원 3,000원 (커피 1잔) 기준:**

```
후원 금액:       3,000원
- 플랫폼 수수료:   150원 (5%)
- PG 수수료:       84원 (~2.8%)
= 크리에이터:    2,766원 (92.2%)
```

**후원 10,000원 기준:**

```
후원 금액:      10,000원
- 플랫폼 수수료:   500원 (5%)
- PG 수수료:      280원 (~2.8%)
= 크리에이터:   9,220원 (92.2%)
```

### 고정 비용 (월)

| 항목 | 비용 |
|------|------|
| 서버 (Vercel Pro) | $20 (~30,000원) |
| DB (Supabase Pro) | $25 (~35,000원) |
| 도메인 | ~2,000원 |
| 이메일 (Resend) | $20 (~30,000원) |
| **합계** | ~100,000원 |

### 변동 비용

| 항목 | 비용 |
|------|------|
| PG 수수료 | 거래액의 ~2.8% |
| 이메일 (초과분) | 건당 |
| 고객 지원 | 시간 투자 |

### 손익분기점

```
고정비용: 100,000원/월
수수료율: 5%
손익분기 거래액: 100,000 / 0.05 = 2,000,000원/월

→ 월 200만원 거래 시 손익분기
→ 활성 크리에이터 40명 × 월 5만원 후원
```

## 가격 정책

### 왜 5%인가?

1. **Buy Me a Coffee와 동일**: 글로벌 스탠다드
2. **투네이션/트윕보다 저렴**: 국내 경쟁력
3. **지속 가능한 사업**: 서버비/운영비 커버
4. **크리에이터 친화적**: 92% 이상 수령

### 가격 정책 원칙

1. **투명함**: 숨겨진 비용 없음
2. **단순함**: 5% 하나만
3. **공정함**: 거래 없으면 비용 없음
