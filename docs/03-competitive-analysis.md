# 경쟁 분석

## 해외 서비스

### Buy Me a Coffee

| 항목 | 내용 |
|------|------|
| **URL** | buymeacoffee.com |
| **설립** | 2018년 |
| **연매출** | ~$5M (2024) |
| **직원** | 30명 |
| **수수료** | 5% + 결제 수수료 (~3%) |

#### 기능
- 일회성 후원 (팁)
- 멤버십 (월/연 구독)
- 샵 (디지털 제품 판매)
- 이메일 뉴스레터
- 모바일 앱
- 분석 대시보드
- GitHub FUNDING.yml 공식 지원

#### 장점
- 심플한 UX
- 낮은 수수료 (5%)
- 개발자 친화적 (GitHub 연동)
- 24/7 지원

#### 한국 사용 시 단점
- 달러 결제만 가능
- PayPal/Stripe 정산 (해외 계좌 필요)
- 한국 후원자 결제 허들 높음
- 환전 수수료 발생

### Ko-fi

| 항목 | 내용 |
|------|------|
| **URL** | ko-fi.com |
| **수수료** | 0% (후원) / 5% (멤버십, 샵) |

#### 기능
- 일회성 후원 (0% 수수료!)
- 멤버십
- 샵 (디지털/물리 제품)
- 커미션
- 라이브 스트리밍 연동

#### 장점
- 후원에 수수료 없음
- 즉시 정산 (PayPal/Stripe)

#### 한국 사용 시 단점
- 마찬가지로 해외 결제/정산
- PayPal 계정 필요

## 우리의 포지션

**"개발자/블로거를 위한 한국형 Buy Me a Coffee"**

| 차별점 | 설명 |
|--------|------|
| **한국 결제** | 카카오페이, 토스페이 |
| **원화 정산** | 국내 계좌 직접 입금 |
| **심플함** | 후원 기능에 집중, 복잡한 기능 X |
| **개발자 친화** | GitHub 뱃지, 마크다운 지원 |
| **누구나 가입** | 심사 없음, 즉시 시작 |

## SWOT 분석

### Strengths (강점)
- 한국 결제 시스템 (카카오페이, 토스)
- 개발자/블로거 특화
- 낮은 수수료 (5%)
- 심플한 UX

### Weaknesses (약점)
- 신규 서비스 (인지도 없음)
- 초기 사용자 확보 필요
- 1인 개발 리소스 한계

### Opportunities (기회)
- 한국 크리에이터 시장 성장 (5.5조)
- 개발자 후원 문화 확산
- 경쟁 서비스 부재

### Threats (위협)
- Buy Me a Coffee 한국 진출 가능성
- 카카오/네이버 직접 서비스 가능성
- 결제사 정책 변경 리스크
