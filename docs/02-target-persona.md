# 02. 타겟 페르소나

## 크리에이터 페르소나

### 페르소나 A: 오픈소스 개발자

```yaml
이름: 김민수
나이: 28세
직업: 스타트업 백엔드 개발자 (4년차)
거주지: 서울
연봉: 6,000만원

활동:
  - GitHub 오픈소스 라이브러리 운영 (Star 500+)
  - 주말마다 사이드 프로젝트 개발
  - 기술 블로그 월 1-2회 포스팅

현재_후원_상황:
  - README에 Buy Me a Coffee 뱃지 있음
  - 월 1-2건 후원 (달러로 받아서 불편)
  - PayPal 환전 수수료 아까움

Pain_Points:
  - "한국 분들이 해외결제 해야 해서 후원 허들이 높음"
  - "GitHub Sponsors 세금 서류가 너무 복잡함"
  - "카카오페이로 받을 수 있으면 좋겠음"

원하는_것:
  - 카카오페이로 간편하게 후원 받기
  - 원화로 국내 계좌 정산
  - GitHub README에 뱃지 달기

예상_후원_규모:
  월_건수: 5-10건
  건당_금액: 3,000-10,000원
  월_예상: 30,000-100,000원
```

```yaml
  - 티스토리 기술 블로그 3년 운영

원하는_것:

  - 글 하단에 "도움됐다면 커피 한 잔" 버튼
  - 월 추가 수익 5-10만원

예상_후원_규모:
  월_건수: 10-20건
  건당_금액: 3,000-5,000원
  월_예상: 50,000-100,000원
```
