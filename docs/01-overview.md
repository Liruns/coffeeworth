# 01. 서비스 개요

## 기본 정보

| 항목 | 값 |
|------|-----|
| 서비스명 | **커피값좀** |
| 영문명 | coffeeworth |
| 한 줄 소개 | 한국 개발자와 블로거를 위한 가장 쉬운 후원 플랫폼 |
| 슬로건 | "당신의 콘텐츠에 커피값좀 보태드릴게요" |

## 문제 정의

### 해외 서비스의 한계
| 서비스 | 문제점 |
|--------|--------|
| Buy Me a Coffee | 달러 결제, PayPal/Stripe 정산, 환전 수수료 |
| Ko-fi | 달러 결제, PayPal 필요 |
| GitHub Sponsors | 미국 세금 서류(W-8BEN), 복잡한 설정 |

### 국내 서비스의 한계
| 서비스 | 문제점 |
|--------|--------|
| 투네이션/트윕 | 스트리머 전용, TTS 등 불필요한 기능 |
| 티스토리 응원하기 | 스토리 크리에이터(5천명)만 가능 |
| 브런치 응원하기 | 브런치 작가만 가능 |

### 빈 틈
- 개발자/블로거용 심플한 후원 플랫폼 = **없음**
- 카카오페이/토스 + 원화 정산 = **없음**
- GitHub 연동 + 한국 결제 = **없음**

## 솔루션

### 핵심 가치 제안
```
"커피값좀"은 한국 개발자와 블로거가
카카오페이/토스로 간편하게 후원받고
원화로 국내 계좌에 정산받는 서비스입니다.
```

### 차별점
| vs | 커피값좀의 차별점 |
|----|------------------|
| Buy Me a Coffee | 카카오페이/토스 결제, 원화 정산 |
| 투네이션/트윕 | 개발자/블로거 특화, 심플함 |
| 티스토리 응원하기 | 누구나 가입, 플랫폼 제약 없음 |

## 타겟 시장

### 1차 타겟 (MVP)
- 오픈소스 개발자 (GitHub 활동)
- 기술 블로거 (티스토리, Velog, 개인 블로그)
- 개발 뉴스레터 작가

### 시장 규모
| 지표 | 수치 | 출처 |
|------|------|------|
| 한국 크리에이터 시장 | 5.5조원 (2024) | 방통위 |
| 전년 대비 성장률 | 4.4% | 방통위 |
| 종사자 수 | 43,717명 | 방통위 |

### TAM/SAM/SOM
| 구분 | 정의 | 추정 규모 |
|------|------|----------|
| TAM | 전체 크리에이터 시장 | 5.5조원 |
| SAM | 개발자/블로거 후원 시장 | 500억원 |
| SOM | 1년 내 확보 가능 | 5억원 |

## 핵심 지표 (MVP 목표)

### 6개월 목표
| 지표 | 목표 |
|------|------|
| 가입 크리에이터 | 1,000명 |
| 활성 크리에이터 | 100명 (10%) |
| 월 거래액 (GMV) | 500만원 |
| 월 수익 | 25만원 |

### 성공 기준
- [ ] 크리에이터 100명이 실제로 후원을 받음
- [ ] 재후원율 20% 이상
- [ ] NPS 50 이상
