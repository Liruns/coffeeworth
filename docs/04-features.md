# 기능 정의

## MVP (v1.0)

> 목표: 4주 내 개발, 핵심 기능만 집중

### 1. 인증/계정

#### 1.1 회원가입/로그인
| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 카카오 로그인 | OAuth 2.0 | P0 |
| 이메일 로그인 | 이메일 + 비밀번호 | P1 |
| 구글 로그인 | OAuth 2.0 | P2 |

#### 1.2 프로필 설정
| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 닉네임 | 표시 이름 | P0 |
| 프로필 이미지 | 업로드 또는 URL | P0 |
| 자기소개 | 짧은 소개글 (500자) | P0 |
| 커스텀 URL | coffee.kr/{username} | P0 |
| 소셜 링크 | GitHub, 블로그, 트위터 등 | P1 |

---

### 2. 크리에이터 페이지

#### 2.1 페이지 구성
```
+------------------------------------------+
|  [프로필 이미지]                           |
|  닉네임                                   |
|  자기소개                                  |
|                                          |
|  [GitHub] [블로그] [트위터]                |
|                                          |
|  +------------------------------------+  |
|  |  ☕ 커피 한 잔 사주기                 |  |
|  |                                    |  |
|  |  [1잔 3,000원] [3잔] [5잔] [직접입력] |  |
|  |                                    |  |
|  |  응원 메시지 (선택)                   |  |
|  |  [____________________________]    |  |
|  |                                    |  |
|  |  [☕ 후원하기]                       |  |
|  +------------------------------------+  |
|                                          |
|  최근 후원자                              |
|  - 익명 님이 커피 1잔 후원 (1시간 전)       |
|  - 홍길동 님이 커피 3잔 후원 (어제)         |
+------------------------------------------+
```

#### 2.2 커스터마이징
| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 커피 가격 설정 | 기본 3,000원, 변경 가능 | P0 |
| 테마 색상 | 프라이머리 색상 선택 | P1 |
| 커피 아이콘 변경 | 이모지 선택 | P2 |

---

### 3. 후원 기능

#### 3.1 후원 프로세스
```
1. 크리에이터 페이지 방문
2. 커피 개수 선택 (1잔/3잔/5잔/직접입력)
3. 응원 메시지 작성 (선택)
4. 결제 수단 선택
5. 결제 진행
6. 완료 화면 + 감사 메시지
```

#### 3.2 결제
| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 카카오페이 | 간편결제 | P0 |
| 토스페이 | 간편결제 | P0 |
| 신용카드 | 토스페이먼츠 | P1 |

#### 3.3 후원자 경험
| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 비회원 후원 | 로그인 없이 결제만 | P0 |
| 익명 후원 | 이름 비공개 옵션 | P0 |
| 후원 메시지 | 텍스트 메시지 (200자) | P0 |

### 5. 연동 도구

## 기능 우선순위 매트릭스

```
        높은 가치
            |
    P0      |      P1
  (MVP 필수) |   (MVP 있으면 좋음)
            |
낮은 노력 --|--------------- 높은 노력
            |
    P2      |      P3
 (v2.0)     |   (나중에)
            |
        낮은 가치
```

### P0 (MVP 필수)
- 카카오 로그인
- 크리에이터 페이지
- 후원 기능 (카카오페이, 토스)
- 기본 대시보드
- GitHub 뱃지

### P1 (MVP 있으면 좋음)
- 이메일 로그인
- 테마 색상
- 신용카드 결제
- 필터링

### P2 (v2.0)
- 구글 로그인
- 커피 아이콘 변경
- 엑셀 다운로드
- 멤버십

### P3 (나중에)
- Slack/Discord 연동
- 분석 기능
- Webhook
